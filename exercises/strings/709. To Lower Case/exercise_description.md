## Problem Description

Given a string `s`, return _the string after replacing every uppercase letter with the same lowercase letter_.

## Examples

- **Example 1**:

  - Input: s = "Hello"
  - Output: "hello"

- **Example 2**:

  - **Input**: s = "here"
  - **Output**: "here"

- **Example 3**:
  - **Input**: s = "LOVELY"
  - **Output**: "lovely"

## Constraints

- $1 \leq s.length \leq 100$
- `s` consists of printable ASCII characters.
